# HR Analytics Dashboard Project | Power BI + SQL

A portfolio-ready Workforce Intelligence and Attrition Prediction Dashboard built with MySQL, Power BI, DAX, and HR analytics storytelling.

## Project Objective
Analyze employee demographics, attendance, performance, salary, attrition, and HR action areas to support better workforce decisions.

## Dashboard Pages
1. Executive Workforce Overview
2. Employee Health Score Dashboard
3. Attrition Risk Analysis Dashboard
4. HR Business Insights and Action Center

## Dataset Tables
- departments.csv
- employees.csv
- attendance.csv
- performance.csv
- attrition_reasons.csv

## Key Business Problems Solved
- Which departments have high attrition risk?
- Which employees need training?
- Which employees are promotion candidates?
- Which employees may leave due to low salary, low attendance, or low performance?
- Which departments are most productive?

## Tools Used
- MySQL Workbench
- Power BI Desktop
- SQL
- DAX
- Data Modeling
- Dashboard Design

## How to Use
1. Open MySQL Workbench.
2. Run `SQL/schema.sql`.
3. Import CSV files from the Dataset folder into their matching tables.
4. Run `SQL/analytics_queries.sql` for analysis.
5. Connect Power BI to MySQL or import CSV files directly.
6. Create relationships and DAX measures.
7. Build the dashboard pages.
